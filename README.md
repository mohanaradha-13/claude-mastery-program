# 🧠 Claude Ecosystem Mastery Program

A free, open-source, self-paced **8-month curriculum** to master the entire Claude AI ecosystem — from absolute beginner to AI Engineer.

**[▶ Launch the Dashboard](https://yourusername.github.io/claude-mastery-program/)**

|-------|-------|------------|

## Features

- **Progress tracking** — check off topics as you complete them, saved in your browser
- **Export / Import** — download your progress as JSON, restore it on any device
- **192 daily topics** across 32 weeks
- **32 hands-on projects** — one per week
- **40 assessments** — weekly quizzes + monthly exams + final comprehensive exam
- **Daily study plan** — optimized for 2 hours/day, 5 days/week
- **Mobile-friendly** — works on phone, tablet, and desktop
- **No login required** — fully client-side, zero tracking

## Quick Start

### Option 1: Use it online (GitHub Pages)

1. Fork this repository
2. Go to **Settings → Pages → Source → Deploy from a branch → `main` → `/ (root)`**
3. Wait ~60 seconds, then visit `https://yourusername.github.io/claude-mastery-program/`

### Option 2: Run it locally

```bash
# Clone the repo
git clone https://github.com/yourusername/claude-mastery-program.git

# Open the file in your browser
open index.html
# or on Linux:
xdg-open index.html
# or on Windows:
start index.html
```

That's it — it's a single HTML file with zero dependencies.

## How Progress Works

- Your progress is saved in **your browser's localStorage** (private to your device)
- Click any topic checkbox to mark it complete
- Use **Export** to download a `.json` backup of your progress
- Use **Import** to restore progress on a new device or browser
- Use **Reset** to start fresh

Each user's progress is completely independent — there's no server, no database, no accounts.

## Study Plan

The curriculum is designed for **2 hours per day**:

| Block | Duration | Activity |
|-------|----------|----------|
| 📖 Concept Learning | 30 min | Read docs, watch tutorials |
| ⚡ Hands-on Practice | 40 min | Build, code, experiment |
| 📝 Exercises | 25 min | Complete assigned tasks |
| 🔄 Revision | 15 min | Summarize key takeaways |
| 🧪 Quiz | 10 min | Self-check questions |

## Assessments

- **Weekly**: 20 MCQs (beginner + intermediate + scenario-based)
- **Monthly**: 50 MCQs (mixed difficulty + real-world problems)
- **Final**: 100 MCQs (comprehensive across all 8 months)
- **Projects**: 3 per month (beginner + intermediate + advanced)

## Tech Stack

This project is intentionally simple:

- **Single HTML file** — no build step, no framework, no npm
- **Vanilla JavaScript** — no React, no dependencies
- **localStorage** — offline-first, zero backend
- **Google Fonts (Inter)** — single external dependency
- **< 40 KB total** — loads instantly everywhere
